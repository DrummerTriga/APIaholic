# APIaholic 🚀🤓

A hands-on frontend project to practice fetching and displaying data from a variety of free and open APIs. Built for learning and experimentation, this project combines practical frontend skills with a playful approach to exploring APIs — after all, consuming APIs is good for our health! 🦾

Check it out LIVE now: [apiaholic.gilsantos-dev.com](https://apiaholic.gilsantos-dev.com/)

## ⚡ Status

**Under Development** – currently experimenting with different APIs and frontend layouts. Things might break… or turn out amazing!

---

## 🛠️ Tech Stack

- **React** (Functional Components & Hooks)
- **TypeScript** (for safe and fun coding!)
- **Axios** (for API requests)
- **TailwindCSS** (for styling, optional)
- Various free & open APIs for practice

---

## 🔍 Features (Planned / WIP)

- Fetch and display data from multiple APIs
- Dynamic rendering of API objects/arrays
- Interactive links and lists
- TypeScript-safe API consumption
- Fun, playful frontend components

---

## 🚀 Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/APIaholic.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```
